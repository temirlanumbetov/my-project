# 🚀 CI/CD Pipeline для автоматического развертывания веб-приложения

Практический DevOps-проект с использованием Jenkins, Ansible, Docker и Bash.

## Используемые технологии

- Jenkins
- Ansible
- Docker
- Ubuntu
- Bash
- Apache2

## Этапы Pipeline

1. Получение кода из GitHub
2. Установка Docker через Ansible
3. Сборка Docker image
4. Запуск контейнера Apache
5. Генерация отчёта CPU/RAM
